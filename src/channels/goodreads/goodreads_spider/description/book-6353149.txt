6353149:::::Luceafarul:::::Luceaf rul este una dintre cele mai celebre poezii ale lui Mihai Eminescu, nceput nc n 1873, dar scris i finisat de-a lungul multor ani p n la publicarea sa n aprilie 1883 n Almanahul societ ii studen e ti Rom nia Jun din Viena. Dup publicarea la Viena, poemul a fost reluat apoi n acela i an n revista Convorbiri literare i n final n volumul princeps intitulat Poesii sub ngrijirea lui Titu Maiorescu. Academia Recordurilor Mondiale (World Records Academy) a anun at n ianuarie 2009 c Luceaf rul este cel mai lung poem de dragoste.
